# Pet Context Badge + Empty-State UX v1

## 1. 목적
- 통계/목록이 어떤 반려견 기준인지 화면에서 즉시 이해되도록 명시한다.
- 선택 반려견 필터 결과가 0건일 때 이탈 없이 `전체 기록 보기`로 전환하게 한다.

## 2. 적용 화면
- Home
  - 주간 통계 상단 컨텍스트 배너
  - 목표 카드 상단 컨텍스트 배지
  - 선택 반려견 필터 0건 시 빈 상태 카드 + `전체 기록 보기`
- WalkList
  - 상단 컨텍스트 배지
  - 선택 반려견 필터 0건 시 빈 상태 카드 + `전체 기록 보기`
  - 전체 기록 모드에서 `기준으로 돌아가기` 버튼 노출

## 3. 상태 규칙
- 기본 모드: `선택 반려견 기준`
- 전환 모드: `전체 기록 보기 모드`
- 자동 복귀:
  - 반려견 선택 변경 시 항상 선택 반려견 기준으로 복귀
  - 앱 재진입 시 선택 반려견 기준 우선

## 4. 빈 상태 판정
- 조건:
  - 전체 기록은 존재
  - 태깅된 산책 기록도 존재
  - 그러나 현재 선택 반려견에 매칭되는 기록은 0건
- 안내:
  - "선택 반려견 기록이 아직 없어요"
  - CTA: `전체 기록 보기`

## 5. 접근성
- 컨텍스트 배지 라벨에 현재 기준/반려견명을 포함
- CTA 버튼 라벨을 `전체 기록 보기`, `선택 반려견 기준으로 돌아가기`로 명시

## 6. 완료 기준
- Home/WalkList 모두 기준 배지 노출
- 선택 반려견 0건 상태에서 CTA로 즉시 전환 가능
- 전환 모드에서 원복 버튼 제공
