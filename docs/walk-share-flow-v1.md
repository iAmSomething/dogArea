# Walk Share Flow v1

## 목적
- 산책 종료/산책 상세 화면에서 결과를 바로 공유한다.
- 카카오톡/인스타그램 전용 SDK 없이 iOS 시스템 공유 시트를 사용한다.

## 범위
- 화면
  - `WalkDetailView` (산책 종료 직후)
  - `WalkListDetailView` (기록 상세)
- 공유 포맷
  - 요약 텍스트 + 지도 이미지(있을 때만 첨부)

## 공유 텍스트 규격
- 헤더: `DogArea 산책 기록`
- 필드
  - 산책 날짜/시간
  - 산책 시간
  - 영역 넓이(㎡ + 평)
  - 영역 포인트 개수
  - 반려견 이름(있을 때)

## UX 규칙
- 지도 이미지가 없어도 텍스트만 공유 가능해야 한다.
- 공유 완료/취소 후 화면 상태가 깨지지 않아야 한다.
- 기존 `사진으로 저장하기` 버튼은 유지한다.

## 비범위(v2)
- 카카오 SDK 직접 연동
- 인스타 스토리 전용 딥링크
- 공유 카드 전용 디자인 템플릿

## QA 체크
- [ ] 산책 종료 직후 화면에서 공유 시트가 열린다.
- [ ] 산책 상세 화면에서 공유 시트가 열린다.
- [ ] 지도 이미지가 없을 때도 텍스트 공유가 된다.
- [ ] 카카오톡/인스타그램 앱이 설치된 경우 시스템 공유 목록에서 선택 가능하다.
