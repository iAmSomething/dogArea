# Walk Point Auto Record v1

## 1. 목적
산책 중 영역 포인트 입력의 수동 탭 의존도를 낮추기 위해 자동 기록 모드를 도입한다.

## 2. 모드
- `수동 기록(manual)`: 기존처럼 사용자가 `+` 버튼 탭으로 포인트를 추가
- `자동 기록(auto)`: 위치 업데이트를 기반으로 포인트를 자동 추가

## 3. 자동 기록 규칙(v1)
- 산책 중(`isWalking = true`)일 때만 동작
- 자동 모드에서 첫 유효 위치는 즉시 1회 기록
- 이후는 아래 조건을 모두 만족할 때 기록
  - 마지막 자동 기록 지점 대비 이동 거리 `>= 12m`
  - 마지막 자동 기록 시점 대비 시간 `>= 8초`
- 노이즈 필터
  - 마지막 포인트 대비 거리 `< 4m`이면 기록하지 않음

## 4. 사용자 제어
- Map 설정에서 `포인트 자동 기록 / 포인트 수동 기록` 토글 제공
- 자동 모드에서도 `+` 버튼 수동 입력은 유지(강제 마킹 보정용)

## 5. watchOS 정합성
- Watch `addPoint`는 수동 우선 이벤트로 유지
- 자동 모드 여부와 무관하게 Watch 수동 입력은 즉시 반영

## 6. QA 체크 포인트
- 자동 모드 ON: 걷기 시작 후 포인트가 주기적으로 누적되는지
- 자동 모드 OFF: 자동 누적 없이 수동 입력만 반영되는지
- 자동 모드 ON + `+` 탭: 수동 포인트가 추가로 반영되는지
- 짧은 제자리 떨림(GPS noise)에서 과도한 포인트 생성이 억제되는지
