# First Walk Onboarding Step2 Recording Sharing v1

- Issue: #706
- Relates to: #271, #266

## 목표
- Step1에서 산책 의미를 이해한 직후, 사용자가 핵심 설정 2가지만 짧게 결정하게 만든다.
- 이후 설정 탭에서 같은 값을 다시 찾고 바꿀 수 있는 재진입 경로를 정의한다.

## Step2에서 다루는 설정 축
1. 기록 방식
2. 공유 기본값

이 두 축 외의 설정은 Step2에서 다루지 않는다.

## 기본값

### 기록 방식 기본값
- 기본값: `수동 포인트 기록`
- 근거
  - 현재 저장소 기본 raw value는 `manual`
  - 첫 사용자가 의도하지 않은 자동 포인트 누적을 바로 겪지 않게 하는 쪽이 안전하다

### 공유 기본값
- 기본값: `OFF`
- 근거
  - 현재 앱 기본 저장값은 `false`
  - 공유는 명시 동의 기반이어야 하며, 기본 ON처럼 보이면 안 된다

## Step2 화면 구조

### 헤더
- 제목: `이제 시작 전에 딱 두 가지만 정할게요`
- 보조 문구: `나중에 설정에서 다시 바꿀 수 있어요.`

### 카드 1. 기록 방식
- 옵션 A: `수동으로 포인트 남기기`
  - 보조 설명: `필요할 때만 직접 찍어 영역을 남겨요.`
- 옵션 B: `걸을 때 자동으로 포인트 남기기`
  - 보조 설명: `걷는 동안 포인트가 자동으로 쌓여요.`
- 기본 선택: `수동`
- 마이크로 카피
  - `처음에는 수동이 더 예측 가능해요`

### 카드 2. 공유 기본값
- 옵션 A: `기본은 비공개로 시작`
  - 보조 설명: `원할 때만 공유를 켜고 다시 끌 수 있어요.`
- 옵션 B: 없음
  - Step2에서는 공유 ON을 기본 추천값처럼 병렬 배치하지 않는다.
- 설명 블록
  - `공유 기능은 동의 후에만 켤 수 있어요`
  - `권한 요청은 실제 공유를 켤 때 따로 안내돼요`

## 스킵 규칙
- Step2는 스킵 가능하다.
- 스킵 시 적용값
  - 기록 방식: `수동 포인트 기록`
  - 공유 기본값: `OFF`
- 스킵 시 사용자 문구
  - `지금은 안전한 기본값으로 시작할게요. 나중에 설정에서 바꿀 수 있어요.`

## 권한 요청과 설정 선택의 분리
- Step2는 `선택 저장` 단계다.
- Step2 안에서 OS 위치 권한 / 공유 권한 요청을 바로 띄우지 않는다.
- 권한 요청은 아래 실제 행동 시점으로 미룬다.
  - 산책 시작
  - 공유 기능 ON 시도

## 선택 결과가 이후 흐름에 연결되는 방식

### 기록 방식
- 지도 탭의 산책 시작/진행/영역 추가 흐름에 바로 반영된다.
- canonical re-entry route
  - `설정 탭 > 산책과 기록 > 포인트 기록 방식`
- 빠른 접근 shortcut
  - `지도 설정 시트 > 기록 방식 토글`

### 공유 기본값
- 라이벌/근처 공유 기능 진입 시 기본 상태를 결정한다.
- canonical re-entry route
  - `설정 탭 > 프라이버시 센터 > 공유 기본값`
- 구현 연결 메모
  - 프라이버시 센터 허브 surface는 `#700/#704`에서 구체화한다.

## 재진입 경로 정의

### 설정 탭 canonical IA
- 섹션 1: `산책과 기록`
  - 항목: `포인트 기록 방식`
- 섹션 2: `프라이버시 센터`
  - 항목: `공유 기본값`
  - 항목: `공유 상태`
  - 항목: `보존/삭제 요청`

### 지도/기능 surface shortcut
- 지도 설정 시트
  - 현재 기록 방식 빠른 토글 유지
- 라이벌/공유 surface
  - 공유를 처음 켜려 할 때 현재 기본값과 동의 상태를 다시 설명

## 금지 규칙
- Step2에서 설정을 2개 이상 늘리지 않는다.
- 공유를 `추천 ON`처럼 보이게 만드는 2열 대칭 선택 UI를 쓰지 않는다.
- 권한 요청과 기본값 선택을 한 팝업에 섞지 않는다.

## 구현 완료 판단 기준
- Step2가 길지 않고 2개 결정만 받는다.
- 스킵 시 `manual/OFF`가 명확히 남는다.
- 사용자가 나중에 `설정 탭`에서 같은 값을 다시 찾을 수 있다.
