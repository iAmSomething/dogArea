# Quest Stage3 UX/Reminder v1

## Scope
- 홈 퀘스트 위젯에 `일일/주간` 탭 전환 추가
- 실패/만료 상태에서 다음 행동 제안 카드 노출
- 로컬 알림 기반 `하루 1회` 퀘스트 리마인드(on/off) 제공

## UX 규칙
1. 기본 탭은 `일일`
2. `주간` 탭은 시즌 점수/기여/오늘 완료를 요약
3. 리마인드 토글은 홈 카드 내부에서 직접 제어
4. 리마인드 시간은 `매일 20:00`, 최대 `1회/일`
5. 권한 거부 시 토글은 자동 OFF 복귀 + 안내 문구 표시

## 실패/만료 가이드
- 완료 기준 미달: 행동량 추가 또는 지도 수동 기록 후 재시도
- 연장 만료: 기본 미션 1개 완료를 우선 제안
- 연장 쿨다운: 기본 미션 진행 안내
- 악천후: 실내 대체 미션 우선 안내

## Acceptance 매핑 (#129)

- 퀘스트 카드 진행/남은 조건: 기존 유지 + 탭 구조에 포함
- 완료 축하/즉시 수령: 기존 유지
- 실패/만료 대체 행동: 제안 카드로 명시
- 하루 1회 리마인드 + 비활성화: 로컬 알림 토글로 충족
