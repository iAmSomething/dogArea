# Season Motion Pack v1

## 목표
- 시즌 점수/결과/보호 상태를 텍스트 중심에서 모션 중심 피드백으로 전환한다.
- 모션은 장식이 아니라 상태 전달(점수 증가, 랭크업, 시즌 전환, 보호 적용) 목적에 한정한다.

## 구성 요소
1. 시즌 게이지 액체 모션
- 트리거: 시즌 점수 증가 시점
- 연출: 게이지 fill 애니메이션 + 하이라이트 wave sweep
- 시간: 증가량 비례, 최소 0.22초 ~ 최대 1.0초

2. 시즌 종료 결과 리빌
- 트리거: 주차 변경 감지(주간 리셋)
- 연출: 랭크 -> 기여 횟수 -> Shield 적용 횟수 순차 노출
- 동작: 1회 자동 재생 후 정적 상태 유지, 사용자 닫기 제공

3. Weather Shield 링 애니메이션
- 상태: 악천후 대체 미션 활성(`risk != clear`) 구간
- 연출: 얇은 trim 링 회전(정보 가림 최소화)
- 집계: Shield 적용 횟수는 시즌 요약/결과에 함께 표시

4. 주간 리셋 전환 배너
- 트리거: 시즌 결과 모달 발생 시점
- 연출: 상단 전환 배너 1.1초 노출

5. 시즌 햅틱 패턴
- 점수 증가: light impact
- 랭크업: success + medium impact(2단)
- Shield 적용: rigid impact
- 시즌 리셋: soft impact

## 접근성/전력 정책
- `Reduce Motion` 또는 `저전력 모드`에서 모션을 축소한다.
- 축소 모드에서는 결과 리빌 순차 모션 없이 즉시 정적 렌더링한다.
- 햅틱 강도는 저전력 모드에서 자동 경량화한다.

## UI 영향 경로
- Home 시즌 카드(게이지/Shield/리셋/결과 오버레이)
- Profile 사용자 정보 화면(현재 시즌 요약 노출)
- 공통 햅틱 유틸(`AppHapticFeedback`)

## 수용 기준
- 점수 증가 직후 게이지 변화가 즉시 인지된다.
- 주간 리셋 시 시즌 결과가 순차 노출 후 정적 상태로 유지된다.
- Shield 상태가 카드/결과에 일관되게 노출된다.
- Reduce Motion/저전력 모드에서 과한 모션 없이 동일 정보 전달이 가능하다.
