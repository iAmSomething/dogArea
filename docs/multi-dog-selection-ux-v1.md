# Multi-Dog Selection UX v1

## 1. 목적
단일 사용자 + 다중 반려견 환경에서 "선택 반려견(selected pet)"을 기준으로 홈/설정/산책 시작 경험을 일관되게 만든다.

## 2. 범위
- 선택 반려견 상태 저장/복원
- 홈 탭 선택 반려견 전환 UI
- 설정 탭 선택 반려견 전환 UI
- 지도 탭 산책 시작 시 선택 반려견 문맥 반영
- `pet.first` 하드코딩 제거

## 3. 상태 모델
- 저장 위치: `UserDefaults`
- 키: `selectedPetId`
- 규칙:
  - 선택값이 비어있거나 유효하지 않으면 첫 활성 반려견으로 보정
  - 반려견 목록 변경 시 선택값 재검증

## 4. 화면 반영
1. 홈(Home)
- 반려견 선택 칩 제공
- 선택 변경 시 `selectedPetId` 저장
- 통계/타이틀은 선택 반려견 이름 기준으로 노출

2. 설정(Notification)
- 반려견 선택 칩 제공
- 선택된 반려견의 프로필/캐리커처 상태 표시

3. 산책 시작(Map)
- 시작 시점에 선택 반려견을 확정
- 시작 모달에 반려견 이름 노출
- 산책 진행 중에는 확정된 반려견 이름 유지

## 5. 완료 기준
- 앱 재실행 후 선택 반려견 복원
- 홈/설정/산책 시작 화면 모두 동일 선택 기준 사용
- 주요 경로에서 `pet.first` 하드코딩 제거
