# Home Weather Pet Guidance Sheet v1

## 목적

홈 날씨 카드는 원시 수치를 보여주고, `더보기` 시트는 오늘 날씨와 선택 반려견 상태를 조합한 행동 가이드를 제공합니다.

## 진입

- 진입점: 홈 `지금 날씨 상세` 카드 하단 CTA
- 컴포넌트 식별자: `home.weather.more`
- 시트 식별자: `sheet.home.weatherGuidance`

## 정보 구성

### 상단 요약

- 제목: `오늘 산책 가이드`
- 관측 요약: 체감 온도, 비/바람, 공기질, 관측 갱신 시각
- 선택 반려견 문맥 배지:
  - 연령대
  - 최근 활동량
  - 견종 또는 체형 추정

### 행동 가이드 섹션

항상 아래 3개 섹션을 고정 순서로 노출합니다.

1. `오늘 산책 시 주의`
2. `산책 권장 방식`
3. `실내 대체 추천`

각 섹션은 일반 경고 나열이 아니라 바로 행동으로 옮길 수 있는 문장으로 구성합니다.

## 규칙 입력

- 날씨:
  - 기온
  - 체감 온도
  - 습도
  - 강수 여부/강수량
  - 풍속
  - 미세먼지
- 반려견:
  - 선택 반려견 이름
  - 나이
  - 견종/체형 추정
  - 최근 일일 산책 시간
  - 최근 주간 산책 횟수

## 대표 규칙

- 체감 또는 기온이 `4도 이하`면 저온 노출 축소 가이드
- 강수/강풍이 있으면 짧은 확인 산책 + 젖은 털/발 관리 가이드
- 체감 고온 또는 고습이면 `짧고 자주` 걷기 가이드
- 미세먼지가 나쁘면 실외 체류 축소 + 실내 대체 우선 가이드
- 고위험 악천후 또는 심한 공기질이면 실내 대체 우선
- 소형견/노령견/유년기는 추위·비·바람 규칙을 더 보수적으로 적용

## 프로필 fallback 정책

- 활성 반려견이 없으면 `기본 안전 기준`으로 안내
- 나이/견종 정보가 비어 있으면 fallback notice를 노출
- fallback이어도 세 섹션은 모두 채우고 빈 화면을 만들지 않음

## 카피 원칙

- 의료 판단처럼 단정하지 않음
- 사용자에게 바로 할 행동을 제안
- `위험`, `주의`, `대체`는 제품 안전 가이드 톤으로만 사용
- `질병`, `진단`, `치료` 같은 의료 어휘는 피함

## 테스트 기준

- 홈 카드 CTA 탭 시 시트가 열린다
- 3개 섹션이 모두 보인다
- 프로필 정보가 부족한 상태에서도 fallback 안내가 보인다
- 닫기 버튼으로 시트가 정상 종료된다
