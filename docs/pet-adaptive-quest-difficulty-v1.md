# Pet Adaptive Quest Difficulty v1 (Issue #147)

## 1. 목표
- 선택 반려견의 상태(연령/활동량/산책 빈도)를 반영해 퀘스트 목표 난이도를 자동 조정한다.
- 과도한 난이도 변동을 방지해 완료 경험을 안정화한다.

## 2. 입력 신호
- 연령대: `puppy(<=1)`, `adult(2~9)`, `senior(>=10)`, `unknown`
- 최근 활동량: 최근 14일 일평균 산책 분(min/day)
- 산책 빈도: 최근 28일 주평균 산책 횟수(count/week)

## 3. 난이도 조정 규칙
- 기본 배율: `1.0`
- 연령 가중치
  - 유년기: `-0.08`
  - 노령기: `-0.12`
- 활동량 가중치
  - 저활동(`<20min/day`): `-0.12`
  - 고활동(`>65min/day`): `+0.10`
- 산책 빈도 가중치
  - 낮음(`<3회/week`): `-0.08`
  - 높음(`>10회/week`): `+0.08`
- 최종 배율 범위: `0.75...1.25`
- 일일 최대 변동폭 제한: 전일 배율 대비 `±0.15`

## 4. 쉬운 날 모드 (수동 오버라이드)
- 선택 반려견 기준으로 `일 1회` 활성화 가능
- 활성화 시 당일 미션 보상 `20%` 감액(`reward * 0.8`)
- 난이도/설명 카드에서 상태를 명시적으로 노출

## 5. 다견 기준
- 계산/저장/표시는 모두 `선택 반려견 petId` 기준으로 분리한다.
- 반려견 전환 시 즉시 난이도 배율과 히스토리를 재계산/재표시한다.

## 6. 사용자 설명/히스토리
- 홈 미션 카드에 아래를 노출한다.
  - 현재 배율과 조정 설명 문구
  - 연령/활동/빈도 신호
  - 최근 히스토리(최근 3일)
  - 쉬운 날 모드 상태(활성/미사용)

## 7. 메트릭
- `indoor_mission_difficulty_adjusted`
- `indoor_mission_easy_day_activated`
- `indoor_mission_easy_day_rejected`
