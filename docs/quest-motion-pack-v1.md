# Quest Motion Pack v1

## 목표
- 퀘스트 진행/완료/수령 상태를 짧고 명확한 모션으로 전달한다.
- 모션 축소 환경에서도 동일한 상태 정보를 색/텍스트로 전달한다.

## 구성
1. 퀘스트 카드 스냅
- 완료 가능(행동량 달성) 상태에서 카드 상세를 접고 CTA를 전면 배치한다.
- 모션 축소 또는 Reduce Motion 시 스냅 연출을 비활성화한다.

2. 진행바 증분 애니메이션
- 실제 상태값 변경 시에만 진행률을 애니메이션으로 증가시킨다.
- 증분 시 강조 glow를 짧게 주어 인지성을 높인다.

3. 완료 모달 마이크로 인터랙션
- 완료 시 700ms 이내 축하 모달을 오버레이로 노출한다.
- 빠르게 사라지며 기존 흐름을 막지 않는다.

4. 즉시 수령 버튼 상태 전환
- 대기(완료 확인) -> 수령 가능(즉시 수령) -> 수령 완료로 전환한다.
- 수령 완료 시 색상/아이콘 전환과 짧은 pulse를 적용한다.

5. 퀘스트 햅틱 패턴
- 진행: light impact
- 완료: success notification
- 실패/기준 미달: warning notification

## 접근성/성능
- `accessibilityReduceMotion` 또는 앱 모션 축소 환경에서 모션을 최소화한다.
- 상태 전달 핵심은 텍스트/색상으로 유지한다.
