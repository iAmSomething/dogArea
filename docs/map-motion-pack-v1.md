# Map Motion Pack v1

## 목표
- 지도 상의 점령/확장 행동에 즉각 피드백을 제공해 상태 변화 인지성을 강화한다.
- 모션은 장식이 아니라 정보 전달 목적(점령 성공, 군집 변화, 날씨 상태 전환)에 제한한다.

## 구성 요소
1. 점령 파동 애니메이션
- 트리거: 산책 포인트 append 시점(수동/자동/워치)
- 표현: 원형 확산 + 페이드
- 지속시간: 기본 0.52초, 축소 모션 모드 0.35초

2. 발바닥 트레일
- 대상: 산책 진행 중 최근 포인트
- 윈도우: 최근 5초
- 렌더 제한: 최대 12개

3. 클러스터 분해/결합 모션
- 트리거: 줌 거리 기반 재클러스터링 시 count 변화
- 분해: scale down pulse
- 결합: scale up pulse

4. 날씨 오버레이 크로스페이드
- 데이터 소스: `UserDefaults(weather.risk.level.v1)`
- 리스크: `clear/caution/bad/severe`
- clear -> 오버레이 제거, 기타 -> 색상 틴트 페이드

5. 지도 인터랙션 햅틱
- 성공: 포인트 점령(수동/워치) 시 light impact
- 경고: 런타임 가드/자동종료 경고 시 warning notification
- 과도한 반복 방지를 위해 throttle 적용

## 성능/접근성
- `모션 축소` 토글 제공(`map.motion.reduced`)
- 시스템 `Reduce Motion`가 켜진 경우에도 축소 모드로 동작
- 짧은 모션과 객체 수 제한으로 맵 가독성과 FPS 저하를 완화
