# Hotspot Widget Radius Preset v1

## 목적

핫스팟 위젯에 고정 반경이 아니라 사용자 선택 preset을 도입해, 같은 프라이버시 정책 안에서 관심 범위를 조절한다.

## Preset

### `0.5km`
- 용도: 가장 가까운 익명 흐름 확인
- 유지 정보:
  - 활성도 단계
  - 단계 분포 요약
  - 마지막 업데이트 시각
- 축약 정보:
  - 좌표/개별 카운트 비노출
  - 상세 셀 목록 비노출

### `1km`
- 용도: 기본 주변 범위 확인
- 유지 정보:
  - 활성도 단계
  - 단계 분포 요약
  - 프라이버시 가드 문구
  - 마지막 업데이트 시각
- 축약 정보:
  - 좌표/개별 카운트 비노출

### `3km`
- 용도: 넓은 범위 추세 확인
- 유지 정보:
  - 활성도 단계
  - 단계 분포 요약
  - stale / delay 중요 문구
  - 마지막 업데이트 시각
- 축약 정보:
  - 좌표/개별 카운트 비노출
  - 넓은 범위일수록 지연 영향이 커지므로 stale 문구를 강화

## 프라이버시 규칙

- 정밀 좌표는 노출하지 않는다.
- 개별 사용자 수치/개별 카운트는 위젯에서 노출하지 않는다.
- `k_anon`, `sensitive_mask`, `percentile_only` 정책은 반경과 무관하게 유지한다.

## 앱 상세 연계

- 위젯 탭 시 앱은 Rival 탭으로 이동한다.
- 이동 시 위젯 preset을 함께 전달한다.
- Rival 상세는 같은 preset을 선택 상태로 열고, 배너로 유입 문맥을 안내한다.

## 동기화 정책

- 앱은 활성화 시 `0.5km`, `1km`, `3km` 3개 preset의 hotspot widget snapshot을 모두 동기화한다.
- 위젯은 선택된 preset에 맞는 snapshot만 읽는다.
