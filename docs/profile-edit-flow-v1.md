# Profile Edit Flow v1 (Existing Users)

## 대상 이슈
- #113 기존 사용자 프로필 필드 편집 플로우 추가

## 목적
- 기존 가입 사용자가 `profileMessage`, `breed`, `ageYears`, `gender`를 설정 화면에서 수정할 수 있게 한다.

## 적용 화면
- `NotificationCenterView` (사용자 정보 화면)

## 편집 범위
- 사용자: 프로필 메시지
- 반려견(선택된 반려견 기준):
  - 품종
  - 나이(0~30)
  - 성별(미지정/수컷/암컷)

## 동작 규칙
- 저장 시 trim 적용
- 나이는 숫자 0~30만 허용
- 빈 값은 nil로 저장
- 저장 성공 시 즉시 로컬 상태 반영 + 선택 반려견 동기화 알림

## 비범위
- 원격 Supabase 동기화
- 반려견 다건 일괄 편집 UI
