# Profile Edit Flow v1 (Existing Users)

## 대상 이슈
- #113 기존 사용자 프로필 필드 편집 플로우 추가
- #376 설정/프로필 편집을 반려견 관리 유스케이스 기준으로 재설계

## 목적
- 기존 가입 사용자가 설정 화면에서 사용자 프로필과 선택 반려견 정보를 즉시 수정할 수 있게 한다.
- 반려견 관리(추가/대표 변경/비활성/재활성)를 설정 화면에서 닫는다.
- 온보딩과 기존 사용자 편집이 동일한 입력 카드와 동일한 검증 규칙을 공유한다.

## 적용 화면
- `NotificationCenterView` (설정/사용자 정보 화면)
- `ProfileFieldEditSheet`
- `PetManagementSheet`
- `ProfileSettingsView`
- `PetProfileSettingView`

## 편집 범위
- 사용자:
  - 사용자 이름
  - 프로필 메시지
  - 사용자 사진
- 반려견(선택 반려견 기준):
  - 이름
  - 사진
  - 품종
  - 나이(0~30)
  - 성별(미지정/수컷/암컷)
- 반려견 관리:
  - 반려견 추가
  - 대표 반려견 지정
  - 반려견 비활성/재활성

## 동작 규칙
- 저장 시 trim 적용
- 사용자 이름/반려견 이름은 공백만 입력할 수 없음
- 나이는 숫자 0~30만 허용
- 빈 값은 nil로 저장
- `selectedPetId`는 항상 활성 반려견 기준으로 보정
- 비활성 반려견은 홈/지도/목록의 선택 후보에서 제외
- 저장 성공 시 즉시 로컬 상태 반영 + Profile sync outbox enqueue/flush 수행
- 반려견 비활성 시 마지막 활성 반려견 1마리는 반드시 유지
