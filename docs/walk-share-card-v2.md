# Walk Share Card v2

## 목적
- 공유 시트에 원본 지도 스냅샷 대신 채널 친화적인 카드 이미지를 첨부한다.
- iOS 시스템 공유를 유지하면서 인스타그램/카카오톡에서 보기 좋은 정사각형 포맷을 제공한다.

## 출력 규격
- 해상도: `1080 x 1080`
- 구성:
  - 상단: 지도 이미지 썸네일(라운드)
  - 하단: 산책 요약 카드(날짜/시간/넓이/포인트/반려견)
  - 푸터: `#dogarea #산책기록`

## 데이터 필드
- 날짜/시간
- 산책 시간
- 영역 넓이(㎡ + 평)
- 영역 포인트 개수
- 반려견 이름(있는 경우)

## UX 규칙
- 공유 payload는 항상 `요약 텍스트 + 카드 이미지`를 우선 사용한다.
- 지도 원본 이미지가 없으면 텍스트-only fallback을 유지한다.
- 기존 사진 저장 버튼 동작은 유지한다.

## QA 체크
- [ ] 산책 종료 화면 공유 시 카드 이미지(1080x1080)가 첨부된다.
- [ ] 산책 상세 화면 공유 시 카드 이미지(1080x1080)가 첨부된다.
- [ ] 지도 이미지가 없을 때 텍스트-only 공유가 유지된다.
- [ ] 공유 취소/완료 후 화면 토스트/상태가 정상이다.
